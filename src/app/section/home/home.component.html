Actualmente Angular es uno de los Framework que mas desarrolladores utilizan y adoptan para la implememtación de aplicaciones, por ello es conveniente aprender a crear proyectos utilizando esta tecnología que Google a desarrollado.


Para poder iniciar a desarrollar una aplicación con angular es necesario contar con las siguientes herramientas:

Node.js
Node Package Manager npm

Se requiere instalar Node para poder acceder al gestor de paquetes npm
