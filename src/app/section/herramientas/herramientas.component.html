Instalación de Node.js y npm

Lo primero que necesitamos es descargar Node.js para poder acceder al gestor de proyectos Node Package Manager npm, el cual nos ayudara a la administración de dependencias, entre sotras cosas.

Nos dirigimos a su pagina oficial y descargamos.


Instalamos dando siguiente y siguiente...

Nos dirigimos a la consola de comandos y verificamos la instalación consultando la version con el comando:

node -v

y consultamos también la version instalada del npm.

npm -v

Con esto hemos comprobado correctamente la instalación.

Instalación de TypeScript

TypeScript es un lenguaje de programación orientado a objetos, basado en javascript y fuertemente tipado.
Lenguaje desarrollado por Microsoft, mediante el cual se desarrollan aplicaciones con Angular.
Se recomienda realizar una instalación global con la finalidad de que se pueda detectar la sintaxis en tiempo de desarrollo.

para realizar la instalación vamos a la pagina oficial para obtener el comando:




Damos click en download

Antes de ejecutar el siguiente comando validamos si contamos con proxy configurado, esto en caso de que en nuestra empresa se encuentre alguno habilitado npm config list si no aparece ninguno configurado, lo realizamos con los sig comandos: npm config set proxy http://proxy.emp:puerto npm config set https-proxy http://proxy.emp:puerto npm config set http-proxy http://proxy.emp:puerto

Copiamos el comando para instalar y ejecutamos en cmd. npm install -g typescript

En este momento yo ya lo tengo instalado por eso cancele la operación, ademas de que la velocidad de mi internet no es muy bueno.

Comprobamos la version instalada con el comando: tsc -v

Instalación correcta.

Instalación de Angular

Angular CLI es la consola de comandos, el gestor de angular, para crear proyectos, claser, servicios, componentes, etc.
Es necesario tener instalado node.js y componente npm. Nos dirigimos a la pigina oficial d e angular para obtener el comando.

Mediante la consola ingresamos el comando: npm install -g @angular/cli

Compobamos la version de angular mediante el comando: ng version




Una vez instalado todo podemos empezar a crear proyectos con angular, simplemente escojemos el edito de nuestra preferencia.
